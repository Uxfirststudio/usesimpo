API Overview

Simpo API provides a unified platform for payments, fraud detection, and SMS services. Our API is RESTful and uses JSON for requests and responses.

GET https://api.userusesimpo.com/v1/health
Check the status of the Simpo API services.
// Example Request
curl -X GET https://api.simpo.com/v1/health \
  -H "Authorization: Bearer YOUR_API_KEY"
// Example Response
{
  "status": "ok",
  "services": {
    "payments": "online",
    "fraud_detection": "online",
    "sms": "online"
  },
  "timestamp": "2023-10-05T10:30:00Z"
}

Authentication

All API requests must include your API key in the Authorization header.

// Include your API key in all requests
Authorization: Bearer sk_test_51Hz...
Content-Type: application/json

Payments API

Process payments through MTN Mobile Money and Airtel Money in Uganda.

POST https://api.simpo.com/v1/payments
Initiate a payment request.
Parameter Type Required Description
amount integer Yes Amount in UGX
phone string Yes Customer phone number (256... format)
provider string Yes Mobile money provider (mtn or airtel)
reference string Yes Your unique transaction reference
description string No Description of the payment
// Example Request
{
  "amount": 10000,
  "phone": "256772123456",
  "provider": "mtn",
  "reference": "order_12345",
  "description": "Payment for Order #12345"
}
// Example Response
{
  "id": "pay_123456789",
  "status": "pending",
  "amount": 10000,
  "phone": "256772123456",
  "provider": "mtn",
  "reference": "order_12345",
  "created_at": "2023-10-05T10:30:00Z"
}

                        Fraud Detection API
                    

                    
                    
Detect and prevent fraudulent transactions with our AI-powered fraud detection system.

                    
                    

                        

                            POST
                            https://api.simpo.com/v1/fraud/detect
                        

                        

                            

                                Analyze a transaction for potential fraud.
                            

                            
                            

                                
                                    

                                        Parameter
                                        Type
                                        Required
                                        Description
                                    

                                
                                
                                    

                                        transaction_id
                                        string
                                        Yes
                                        Unique transaction identifier
                                    

                                    

                                        amount
                                        integer
                                        Yes
                                        Transaction amount in UGX
                                    

                                    

                                        user_id
                                        string
                                        Yes
                                        Unique user identifier
                                    

                                    

                                        phone
                                        string
                                        Yes
                                        User phone number
                                    

                                    

                                        device_id
                                        string
                                        No
                                        Device fingerprint
                                    

                                    

                                        ip_address
                                        string
                                        No
                                        User IP address
                                    

                                    

                                        location
                                        object
                                        No
                                        Geolocation data
                                    

                                
                            

                            
                            

                                // Example Request

                                {

                                  "transaction_id": "txn_123456789",

                                  "amount": 150000,

                                  "user_id": "user_12345",

                                  "phone": "256772123456",

                                  "device_id": "a13xC34e516g7",

                                  "ip_address": "192.168.1.1",

                                  "location": {

                                    "city": "Kampala",

                                    "country": "UG"

                                  }

                                }
                            

                            
                            

                                // Example Response

                                {

                                  "transaction_id": "txn_123456789",

                                  "risk_score": 0.87,

                                  "risk_level": "high",

                                  "reasons": [

                                    "Unusual transaction amount",

                                    "New device detected"

                                  ],

                                  "recommendation": "review"

                                }

                        SMS API
                    

                    
                    
Send SMS messages to Ugandan phone numbers with our reliable SMS gateway.

                    
                    

                        

                            POST
                            https://api.simpo.com/v1/sms/send
                        

                        

                            

                                Send an SMS message.
                            

                            
                            

                                
                                    

                                        Parameter
                                        Type
                                        Required
                                        Description
                                    

                                
                                
                                    

                                        to
                                        string
                                        Yes
                                        Recipient phone number (256... format)
                                    

                                    

                                        message
                                        string
                                        Yes
                                        SMS message content
                                    

                                    

                                        sender_id
                                        string
                                        No
                                        Sender ID (default: SIMPO)
                                    

                                    

                                        reference
                                        string
                                        No
                                        Your unique reference for this message
                                    

                                
                            

                            
                            

                                // Example Request

                                {

                                  "to": "256772123456",

                                  "message": "Your payment of UGX 10,000 was successful. Ref: ORD12345",

                                  "sender_id": "SIMPO",

                                  "reference": "msg_12345"

                                }
                            

                            
                            

                                // Example Response

                                {

                                  "id": "sms_123456789",

                                  "to": "256772123456",

                                  "message": "Your payment of UGX 10,000 was successful. Ref: ORD12345",

                                  "status": "queued",

                                  "reference": "msg_12345",

                                  "created_at": "2023-10-05T10:30:00Z"

                                }
